# Holberton School - Low-Level Programming

This repository contains all the projects completed during the **Low-Level Programming** curriculum at Holberton School. These projects are focused on mastering the **C programming language**, understanding fundamental concepts like memory management, data structures, and file I/O, and adhering to strict coding standards.

## ‚öôÔ∏è General Requirements

All C projects in this repository adhere to the following standards:

  * [cite\_start]**Environment:** Files are compiled and tested on **Ubuntu 20.04 LTS**[cite: 2559].
  * [cite\_start]**Compiler:** All C files are compiled with the **`gcc`** compiler[cite: 2559].
  * [cite\_start]**Compilation Flags:** Compilation uses the options `-Wall -Werror -Wextra -pedantic -std=gnu89`[cite: 2559].
  * [cite\_start]**Code Style:** The codebase follows the **Betty style** (checked using `betty-style.pl` and `betty-doc.pl`)[cite: 2559].
  * **Code Structure:**
      * [cite\_start]No more than 5 functions per file[cite: 2559].
      * [cite\_start]No global variables are allowed[cite: 2559].
      * [cite\_start]Prototypes for all functions are included in the designated header file for each project (e.g., `main.h` or `lists.h`, etc.)[cite: 2559, 2585].
